Git 사용법 1

Posted on 2022-09-27 In IT , Git/Github

어제부터 오늘 사이에 git을 사용하는 법을 배웠습니다.
저는 windows 사용자이기 때문에 https://gitforwindows.org 라는 곳에서 git을 설치했습니다.

Shell이란 운영체제의 커널과 사용자를 이어주는 소프트웨어입니다. git은 bash라는 shell을 이용하여 작성할 수 있습니다.

Shell의 Command에 대해서 알아보겠습니다.

  1. cd : 폴더에 접근할 때 사용합니다. cd Documents/라는 명령어는 Documents라는 폴더로 이동하겠다~ 라는 뜻입니다. 또한 상위 폴더로 이동하기 위해서도 사용하는데요. cd ..이라는 명령어를 통해 상위 폴더로 이동할 수 있습니다.
  2. mkdir : make a directory라는 뜻입니다. 뜻 그대로 새로운 폴더를 생성합니다.
  3. touch : 새로운 ‘파일’을 생성할 때 사용합니다.
  4. mv : move의 줄임말입니다. 파일을 다른 폴더로 이동할 때나 이름을 바꿀 때 사용합니다.
  5. cp : copy의 줄임말입니다. 파일의 복사본을 생성할 때 사용합니다.
  6. rm : remove의 줄임말입니다. 파일을 삭제할 때 사용합니다.
  7. cat : catenate의 줄임말입니다. 파일을 배쉬 내에서 확일할 때 사용합니다.
  8. vi : vim을 사용해서 파일을 작성할 때 사용합니다.

Git의 장점

  1. 빠른 속도, 단순한 구조
  2. 분산형 저장소 지원
  3. 비선형적 개발(수 천개의 브랜치) 가능
